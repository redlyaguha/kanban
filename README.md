# Kanban Board - Система управления задачами

FastAPI веб-приложение для управления проектами и задачами по методологии Kanban.

## Функциональность

- **Управление пользователями** - регистрация, аутентификация (OAuth2)
- **Управление проектами** - создание, редактирование, удаление проектов
- **Управление задачами** - создание задач, смена статусов, назначение исполнителей
- **Комментарии** - добавление комментариев к задачам
- **Права доступа** - владелец проекта и участники

## Технологии

- **Python 3** - основной язык разработки
- **FastAPI** - современный веб-фреймворк
- **SQLAlchemy** - ORM для работы с базой данных
- **Pydantic** - валидация данных и схемы
- **OAuth2** - аутентификация и авторизация
- **CORS Middleware** - поддержка кросс-доменных запросов

## Модели данных

### User (Пользователь)
- id: int (PK)
- email: str (уникальный)
- hashed_password: str
- is_active: bool
- projects: List[Project] (many-to-many через project_members)

### Project (Проект)
- id: int (PK)
- name: str
- owner_id: int (FK -> users.id)
- members: List[User] (many-to-many)
- tasks: List[Task] (one-to-many)

### Task (Задача)
- id: int (PK)
- title: str
- description: str
- status: str (TODO, IN_PROGRESS, DONE)
- assignee_id: int (FK -> users.id)
- project_id: int (FK -> projects.id)
- comments: List[Comment] (one-to-many)

### Comment (Комментарий)
- id: int (PK)
- text: str
- task_id: int (FK -> tasks.id)
- author_id: int (FK -> users.id)

## API Эндпоинты

### Пользователи
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/users/` | Регистрация нового пользователя |
| POST | `/token` | Получение JWT токена (логин) |
| GET | `/users/me` | Информация о текущем пользователе |

### Проекты
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/projects/` | Создание проекта |
| GET | `/projects/` | Список проектов пользователя |
| GET | `/projects/{id}` | Информация о проекте |
| PUT | `/projects/{id}` | Обновление проекта |
| DELETE | `/projects/{id}` | Удаление проекта |

### Задачи
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/projects/{id}/tasks/` | Создание задачи в проекте |
| GET | `/tasks/{id}` | Информация о задаче |
| PUT | `/tasks/{id}` | Обновление задачи |
| PUT | `/tasks/{id}/status` | Смена статуса задачи |
| DELETE | `/tasks/{id}` | Удаление задачи |

### Комментарии
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/tasks/{id}/comments/` | Добавление комментария |
| GET | `/tasks/{id}/comments/` | Список комментариев задачи |

## Выполненная работа

1. **Настройка FastAPI** - создание приложения с CORS поддержкой
2. **Моделирование БД** - разработка схемы данных (User, Project, Task, Comment)
3. **CRUD операции** - реализация полного цикла работы с данными
4. **Аутентификация** - настройка OAuth2 с JWT токенами
5. **Валидация** - Pydantic схемы для всех входящих данных
6. **Связи** - настройка отношений many-to-many (пользователи и проекты)
7. **Безопасность** - хеширование паролей, проверка прав доступа

## Как запустить

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск приложения
```bash
uvicorn app.main:app --reload
```

### Swagger документация
После запуска документация будет доступна по адресу:
```
http://localhost:8000/docs
```
